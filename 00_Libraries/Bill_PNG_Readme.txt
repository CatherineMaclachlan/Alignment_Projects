Set Up
-------
Download latest version (here, libpng-1.5.6.tar.gz).

Right click in Windows Explorer and select WinZip to unpack.
(In linux NX, just double-click gz file and drag folder to extract).

According to INSTALL document:
> ./configure CFLAGS=-O3 --prefix=/groups/apig/share/ClusterSupport/Libraries/PNG --with-zlib-prefix=/groups/apig/share/ClusterSupport/Libraries/ZLIB

> make check
> make install

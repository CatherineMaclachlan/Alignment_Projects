Set Up
-------
Download latest version (here, zlib-1.2.5.tar.gz).

Right click in Windows Explorer and select WinZip to unpack.
(In linux NX, just double-click gz file and drag folder to extract).

According to zlib-1.2.5/README, follow instructions in Makefile.in:
> CFLAGS=-O3 prefix=/groups/apig/share/ClusterSupport/Libraries/ZLIB ./configure
> make test
> make install prefix=/groups/apig/share/ClusterSupport/Libraries/ZLIB
